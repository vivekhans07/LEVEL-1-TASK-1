# 📚 LearnWell Tutorials – Landing Page

A modern, responsive landing page developed as **Level 1 – Task 1** for the **Oasis Infobyte Web Development & Designing Internship (OIBSIP)**.

## 🚀 Project Overview

This project is a fully responsive landing page for **LearnWell Tutorials**, a fictional coaching institute. The objective was to create an attractive, user-friendly webpage using only **HTML5** and **CSS3**, following modern web design principles.

The landing page showcases the institute's services, subjects offered, testimonials, and a clear call-to-action for student enrollment.

---

## ✨ Features

- Responsive design for desktop, tablet, and mobile
- Modern UI with clean layout
- Sticky navigation bar
- Hero section with statistics
- Subjects section
- "How It Works" section
- Why Choose Us section
- Student testimonials
- Call-to-Action (CTA)
- Professional footer
- Smooth scrolling navigation
- Hover animations and transitions

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Google Fonts (Syne & Inter)
